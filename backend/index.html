VISITS

/api/visits

  • GET: view all visits
  • POST: add new visit(checkin)

    • parameters: host_id, host_name, visitor_name,visitor_email, visitor_no, role

/api/visits/:id :

  • GET: view single visit
  • PUT: edit single visit
  • DELETE : delete single visit

/api/date/visits

  • GET: view visits by date

    • query parameters: date

/api/visits/today

  • GET: view visits for current day
    • >

/api/active/visits

  • GET: view all active visits{haven't checked out)

/api/checkedout/visits

  • GET: view all checkedout visits

/api/visits/checkout/:id

  • PATCH: checkout
