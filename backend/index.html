VISITS

/api/visits

  • GET: view all visits
  • POST: add new visit(checkin)

    • parameters: host_id, host_name, visitor_name,visitor_email, visitor_no, role

/api/visits/:id :

  • GET: view single visit
  • PUT: edit single visit
  • DELETE : delete single visit

/api/active/visits

  • GET: view all active visits{haven't checked out)

/api/visits/checkout/:id

  • PATCH: checkout

VISITORS

/api/visitors

  • GET: view all visitors
  • POST: register a visitor

    • parameters: name, email_id, mobile_no

/api/visitors/:id

  • GET : view single visitor
  • PUT: edit single visitor
  • DELETE: delete single visitor

HOSTS

/api/hosts:

  • GET : view all hosts

    • parameters: name, email_id, mobile_no

/api/hosts/:id

  • GET : view single host
  • PUT: edit single host
  • DELETE: delete single host

    • parameter:password

/api/login/host

  • POST: authenticate host
